# Система расчета зарплаты

## Обзор

Система расчета зарплаты поддерживает:
- **Минимальную ставку** (по умолчанию 1500 ₽)
- **Процент от продаж с бара** (по умолчанию 10%)

## Как это работает

### Формула расчета зарплаты:

```
Базовая зарплата = max(минимальная ставка, обычная ставка × часы)
Итоговая зарплата = Базовая зарплата + (Продажи с бара × Процент / 100)
```

### Примеры:

**Пример 1: Почасовая оплата**
- Ставка: 200 ₽/час
- Отработано: 5 часов
- Минимальная ставка: 1500 ₽
- Продажи с бара: 5000 ₽
- Процент с бара: 10%

Расчет:
- Обычная зарплата: 200 × 5 = 1000 ₽
- Базовая зарплата: max(1500, 1000) = **1500 ₽** (применяется минимум)
- Бонус с бара: 5000 × 10% = 500 ₽
- **Итого: 1500 + 500 = 2000 ₽**

**Пример 2: Оклад**
- Оклад: 30000 ₽
- Минимальная ставка: 1500 ₽
- Продажи с бара: 10000 ₽
- Процент с бара: 10%

Расчет:
- Базовая зарплата: max(1500, 30000) = **30000 ₽**
- Бонус с бара: 10000 × 10% = 1000 ₽
- **Итого: 30000 + 1000 = 31000 ₽**

## Настройка сотрудника

При создании/редактировании сотрудника можно указать:
1. **Тип оплаты**: Оклад или Почасовая ставка
2. **Ставка/Оклад**: Основная ставка
3. **Минимальная ставка**: Минимальная гарантированная сумма (по умолчанию 1500 ₽)
4. **Процент с бара**: Процент от продаж с бара (по умолчанию 10%)

## Учет продаж с бара

### Как привязать продажу к сотруднику:

1. При добавлении дохода выберите категорию **"Бар"**
2. Поле "Сотрудник" автоматически появится
3. Выберите сотрудника, который сделал продажу
4. Укажите сумму (наличными и/или терминалом)
5. Сохраните

### Важно:

- Продажи с бара учитываются только если указан сотрудник
- Процент рассчитывается от суммы продаж за выбранный период
- Продажи учитываются только из категории "Бар"

## Просмотр зарплаты

### Для администратора (сотрудника):
- Раздел "Моя зарплата" показывает:
  - Базовая зарплата
  - Бонус с бара
  - Продажи с бара
  - Итоговая зарплата

### Для владельца:
- Ведомость на выплату показывает детализацию по каждому сотруднику
- Детальная информация о сотруднике показывает статистику

## Настройка по умолчанию

Все сотрудники автоматически получают:
- Минимальная ставка: **1500 ₽**
- Процент с бара: **10%**

Эти значения можно изменить для каждого сотрудника индивидуально.
